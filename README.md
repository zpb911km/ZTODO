# ZTODO

![ZTODO](https://github.com/zpb911km/ZTODO/blob/main/src-tauri/icons/128x128.png)

这是一个使用Tauri和Angular的桌面应用

用来管理一切需要使用时间来组织的事件

适用于从年到分钟的系列时间精度

## 功能

### 日历

日历默认聚焦到当前时间

点击 `(+)` 按钮进入更精细的时间精度

点击 `(-)` 按钮进入更粗略的时间精度

点击 `[|]` 按钮滚动到当前时间

### 事件

可以添加事件，包括标题、描述、时间、标签、重要性等

可以编辑事件

可以删除事件

点击 `[+]` 按钮进入事件添加页面

点击事件可以进入事件编辑页面
